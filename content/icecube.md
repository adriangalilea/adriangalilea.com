🧑‍🍳Cooked a [cli file sharing + link shortener](https://github.com/adriangalilea/share) on Cloudflare's free tier.

Set up a domain like [🧊.to](https://🧊.to)

Then just:

`share upload README.md` → [🧊.to/readme.md](https://🧊.to/readme.md)

`share link https://github.com/adriangalilea/share` → [🧊.to/gh](https://🧊.to/gh)
